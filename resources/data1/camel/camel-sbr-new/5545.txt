Support key password for private key access in XMLSecurity    Enhance the XMLSecurity component to enable access of a private key (for decryption) from a Java key store using a password that is unique to the key  and different than the key store password.The likely solution is to allow an optional 'keyPassword' parameter in the XMLSecurity data format.A description of the issue in context is here :http://camel.465427.n5.nabble.com/XMLSecurity-key-recovery-fails-when-keystore-and-key-use-different-passwords-td5718094.html
