Sending a message to log endpoint should not convert payload to StreamCache    We should not automatic try to convert the payload to a StreamCache if you send a message to a log endpoint. StreamCache newBody = message.getBody(StreamCache.class); if (newBody != null) { message.setBody(newBody); }Only if people have explicit enabled stream cache we can safely log it.Otherwise we should not output streams in the log.We should use the extractBodyForLogging from MessageHelper instead which cater for this.
