<title>Issue 9860 -   chromium -    ChromeHTML URI handler vulnerability - Project Hosting on Google Code</title> <pre> A vulnerability in the ChromeHTML URI handler allows an attacker to bypass  the Same Origin Policy for any site and also enumerate victims files and  directories.    When loaded in Internet Explorer  a specially crafted HTML page can launch  Google Chrome with an arbitrary URI without requiring any user interaction.     A more detailed and technical security advisory explaining the issues is  attached. </pre>
