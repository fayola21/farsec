Using javascript  set the cursor to 'move'  and then set it to '-webkit-  grab' or '-webkit-grabbing'.  2. The '-webkit-grab' and '-webkit-grabbing' cursor styles will cause the   cursor to be displayed as the default pointer.      Normally  setting an invalid style means that the previous style remains.   This makes it possible for multiple browsers to appear correctly without   detecting them exactly. For example  this code will display either arrows   in four directions or an open hand in IE  Firefox  Safari  and several   other browsers. All the browsers support the 'move' cursor  which displays   four arrows  but the open hand is more appropriate for indicating that   dragging is possible but not currently active.                document.body.style.cursor = 'move';              document.body.style.cursor = 'grab';              document.body.style.cursor = '-moz-grab';              document.body.style.cursor = '-webkit-grab';    Even though  '-webkit-grab' is the last value set  it is ignored for other   browsers that don't support it  so they continue to use the last valid   cursor set.
