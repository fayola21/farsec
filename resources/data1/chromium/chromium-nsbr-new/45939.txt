But it shouldn't be like that    as a user  we just visit the XML file and it should render.    According the the spec  the xsl:stylesheet can be:  &lt;xsl:stylesheet version=&quot;1.0&quot;   xmlns:xsl=&quot;<a href="http://www.w3.org/1999/XSL/Transform">http://www.w3.org/1999/XSL/Transform</a>&quot;&gt;     But that will result in a blank page  if we add xmlns=&quot;&quot; attribute there    it works just fine. </pre>
