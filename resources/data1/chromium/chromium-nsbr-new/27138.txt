<title>Issue 27138 -   chromium -    Chrome has poor battery life compared to other browsers? - Project Hosting on Google Code</title> <pre> <a href="http://anandtech.com/mobile/showdoc.aspx?i=3636">http://anandtech.com/mobile/showdoc.aspx?i=3636</a>  describes a benchmark of browser power consumption  (charge and discharge the battery a few times  charge it up   start the browser  create tabs for AnandTech.com and two other sites   reload every 60 seconds  continue until OS says battery is dead).    <a href="http://anandtech.com/mobile/showdoc.aspx?i=3636&amp;p=3">http://anandtech.com/mobile/showdoc.aspx?i=3636&amp;p=3</a>  gives results for a Core 2 Duo system:    IE 8            227 minutes  Firefox+Adblock 222  Chrome 2        218  Firefox 3.5.2   210  Safari 4        184    <a href="http://anandtech.com/mobile/showdoc.aspx?i=3636&amp;p=4">http://anandtech.com/mobile/showdoc.aspx?i=3636&amp;p=4</a>  gives results for an Atom system:    Chrome 2         454 minutes  IE 8             441 minutes  Firefox 3.5.2    439  Firefox+Adblock  438  Safari 4         332    It would be great if Chrome did as well as the other  browsers under both test conditions. </pre>
