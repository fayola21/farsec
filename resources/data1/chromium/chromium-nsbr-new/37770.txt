<title>Issue 37770 -   chromium -    html5 videos on YouTube don't work if you have selected 'Block sites from setting any data' in the Content settings. - Project Hosting on Google Code</title> <pre> Chrome Version       : 5.0.342.2 dev      1. Clear all your cookies.  2. Go to Options &gt; Under the Hood &gt; Content settings  and select 'Block   sites from setting any data'. Also  add 'www.youtube.com' to the exceptions   list. Select 'Allow' for it.  3. Go to <a href="http://www.youtube.com/html5">http://www.youtube.com/html5</a> and join the html5 beta.  4. Click on a video which works with html5  for example:   <a href="http://www.youtube.com/watch?v=0Z1vJrIAg-0">http://www.youtube.com/watch?v=0Z1vJrIAg-0</a>      Video should start playing since 'www.youtube.com' is set to 'Allow' in the   exceptions list.      Video doesn't start playing. Just loads and nothing happens.    Video starts playing after allowing local data to be set and refreshing the   video page.   </pre>
