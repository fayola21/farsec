Create a &lt;fieldset&gt;...&lt;/fieldset&gt; node  2. Within this  create a &lt;legend&gt; Some Text &lt;/legend&gt;  3. Apply text-align:center to the &lt;legend&gt; element from a &lt;style&gt; tag  such as    &lt;style&gt;    legend {      text-align: center;    }  &lt;/style&gt;  ...  &lt;fieldset&gt;    &lt;legend&gt;Legend Text&lt;/legend&gt;  &lt;/fieldset&gt;    What is the expected result? The expected result is that the text within  the legend element should be center aligned  not the legend element. This  is the purpose of imposing text-align on the element.       What happens instead? The &lt;legend&gt; element becomes aligned as well as  aligning the text within it. Imposing &quot;text-align&quot; on an element should  enforce text within that element to be aligned as specified  but not the  element itself.    The element is not text  and should not be aligned.  </pre>
