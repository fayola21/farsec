<title>Issue 38990 -   chromium -    AppCache: anonymous event handlers go missing in action - Project Hosting on Google Code</title> <pre> If the page uses an anonymous event handler  on the applicationCache object  the reference   to the handler can get silently dropped and events won't get delivered into the page.    // With this  the event can go missing.  applicationCache.oncached = function() { appendTextOut('oncached'); }    // Whereas this will always work.  applicationCache.oncached = alwaysWorks;  function alwaysWorks() { appendTextOut('oncached'); }     </pre>
