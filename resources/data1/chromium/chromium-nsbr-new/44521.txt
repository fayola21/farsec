Implement a git post-receive hook that pushes the hash parent relationship.  For each line &quot;&lt;old-value&gt; SP &lt;new-value&gt; SP &lt;ref-name&gt; LF&quot;  git log for each &lt;new-value&gt; until it finds &lt;old-value&gt;. This assumes   history rewriting is disabled. It is disabled for all chromium os git   repositories.    The command to retrieve the log is:    git log &lt;old-value&gt;..&lt;new-value&gt; --pretty=oneline    Push all these &quot;hash -&gt; parent hash&quot; relation to aebuild.
