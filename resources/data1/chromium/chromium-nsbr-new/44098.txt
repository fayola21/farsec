<title>Issue 44098 -   chromium -    Build breaks on concatenated_devtools_js.vcproj if list of dev tools files does not fit command line - Project Hosting on Google Code</title> <pre> concatenated_devtools_js.vcproj invokes concatenate_js_files.py passing the   entire list of JS files required by devtools on the command line. When new   files are added to dev tools  the list hits command line length limit on   Windows (~8K).    The proposed fix is to use file list extracted from devtools.html  as it   already has all files in proper order. The list only has relative filenames    so we're adding search path as a parameter to concatenate_js_files.py.
