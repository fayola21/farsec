<title>Issue 14023 -   chromium -    Help users get a new default browser on uninstall - Project Hosting on Google Code</title> <pre> If Google Chrome is the default browser and you uninstall it  you're left   with no default browser. This can be frustrating.    Restoring the previous default browser might not be possible because the way   we set the default browser on Vista and 7 involves an opaque call into a   blackbox interface. See <a href="http://www.google.com/codesearch/p?">http://www.google.com/codesearch/p?</a>  hl=en&amp;sa=N&amp;cd=1&amp;ct=rc#YwaaF8DorRY/src/chrome/installer/util/shell_util.cc&amp;l=4  99    Since we cannot restore the previous browser  we should help the user pick   another browser as the default.      </pre>
