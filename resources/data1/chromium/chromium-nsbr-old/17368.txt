<title>Issue 17368 -   chromium -    There should be a way to dock DevTools window. - Project Hosting on Google Code</title> <pre> DevTools window can now be docked on Windows and Linux. The way it is  implemented is following:  - browser window has a splitter in it called contents_split  - this splitter hosts main tab contents container and devtools' tab  contents container.  - there are UpdateDevTools and FocusDevTools methods that should be  implemented as in Windows / Linux.    It should be really straightforward  see Windows and Linux CLs that enabled  it on corresponding platforms:    Windows:  <a href="http://codereview.chromium.org/151150">http://codereview.chromium.org/151150</a> - basic implementation  <a href="http://codereview.chromium.org/155423">http://codereview.chromium.org/155423</a> - brings focus to the devtools  contents container.    Linux:  <a href="http://codereview.chromium.org/149675">http://codereview.chromium.org/149675</a> - basic implementation  <a href="http://codereview.chromium.org/149748">http://codereview.chromium.org/149748</a> - storing split location   </pre>
