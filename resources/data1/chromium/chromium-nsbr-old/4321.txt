<title>Issue 4321 -   chromium -    Scons variable to build with system libraries - Project Hosting on Google Code</title> <pre> Here is a patch that allows chromium to be built with some system libs.  It introduces a variable called SYSTEM_LIBS  consisting of a  comma-separated list of system libraries. So far  bzip2  libpng  libjpeg   libxml  libxslt  lzma_sdk  zlib  hunspell  sqlite  libevent are recognized.   </pre>
