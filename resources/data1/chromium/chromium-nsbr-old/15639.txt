<title>Issue 15639 -   chromium -    Cannot change the default Malayalam font for text fields - Project Hosting on Google Code</title>   1. The platform is Windows XP.  2. Install the Malayalam font AnjaliOldLipi from   <a href="http://downloads.sourceforge.net/varamozhi/AnjaliOldLipi.ttf">http://downloads.sourceforge.net/varamozhi/AnjaliOldLipi.ttf</a>  3. Set the font to AnjaliOldLipi for all the font categories like Serif  Sans-  serifc and Fixed Width  4. In Windows Desktop properties &gt; Appearance tab &gt; Advanced button  set   all UI elements to use AnjaliOldLipi.
