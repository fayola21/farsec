<title>Issue 4926 -   chromium -    CppBoundClass getter/setter binding patch - Project Hosting on Google Code</title> <pre> CppBoundClass is limited to binding javascript property objects to instance  members of the class. This patch allows binding a getter and setter to a  property name  i.e.:    class JsObject : public CppBoundClass  {  public:    int x;      JsObject() {      BindProperty(&quot;x&quot;  &amp;JsObject::getX  &amp;JsObject::setX);    }      void getX(const CppArgumentList &amp;args  CppVariant *result) {      result-&gt;Set(x);    }      void setX(const CppArgumentList &amp;args  CppVariant *result) {      int x = args[0].toInt32();      if (x &gt; 0) { this-&gt;x = x; }    }  };    // JsObject instance bound as &quot;object&quot;  object.x = 100;  object.x = -1;  alert(object.x);  // should be 100   </pre>
