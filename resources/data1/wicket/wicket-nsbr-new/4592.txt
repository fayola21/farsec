WicketAjaxDebug.logError used in Wicket.Head.addJavascript - JS error when wicket-ajax-debug.js not loaded    Wicket.Head.addJavascript (defined in wicket-ajax.js) encapsulates javascript in this code:content = 'try{'+content+'}catch(e){WicketAjaxDebug.logError(e);}'; However  WicketAjaxDebug can be undefined as it's declared in wicket-ajax-debug.js which might not be included. As a consequence  catch block will fail and obfuscate any error catched.I believe this catch block should call Wicket.Log.error(e) instead.
