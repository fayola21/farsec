Websocket broadcast support does not work with OSGi    WICKET-4832 provides websocket support but this does not work in an OSGi environment because the packages are not unique.This patch provides the changes to make it run under OSGi. Most of the changes are in the wicket-native-websocket modules  but WicketFilter from wicket-core has a new method to let AbstractUpgradeFilter only override the required parts (was TODO in original code).
