[devutils] Can only inspect pages in same pagemap as the inspector    The InspectorPage presents a list of pagemaps and pages contained within them  with links to inspect the contents of each page ID.However  when one of these is clicked it tries to resolve the page ID within the current pagemap of the inspector page itself  which is quite possibly not the case.Here is a patch which passes the pagemap name alongside the page ID and looks it up appropropriately.
