Conversion Exception Thrown for Submitting A Page with Multiple Forms within A Border    'org.apache.wicket.util.convert.ConversionException: Can't convert null value to a primitive class: int for setting it on' exception thrown when submitting a Page with multiple Forms within a Border. A 'Bordered' Page with a single Form would not have this problem manifests. Also  if the Page by itself (not contained in a Border) is alright too.
