Empty File Upload field breaks validation of other fields in WicketTester.    Submitting form with empty FileUpload -field break validations of other fields in WicketTester: TextField mandatority (field.setRequired(true)). TextField input length validation (field.add(StringValidator(lengthBetween(...)))). DropDownChoice validation does not 'see' selection.If File Upload -field is filled  then validations above work.Form submits manually without a problem in web browser.
