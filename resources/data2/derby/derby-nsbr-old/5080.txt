Utilize JQL to fetch JIRA issue list for release notes generation    DERBY-4857 made it possible to use the JIRA SOAP API to fetch a list of issues fixed in a release. However  the release manager have to manually create a JIRA filter and feed the filter id to the tool.Now that the Apache JIRA instance has been updated to 4.x  we can use the JIRA Query Language (JQL) to avoid that manual step.
