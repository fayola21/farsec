Remove resultSetHoldability property from ClientDataSource    ClientDataSource (through ClientBaseDataSource) implements a Java Bean property resultSetHoldability which is not documented in the functional spec at.http://incubator.apache.org/derby/papers/DerbyClientSpec.htmlJDBC provides standard ways to set the holdability of ResultSets  so a non-standard separate mechanism is not required. The property and associated code needs to be removed.
